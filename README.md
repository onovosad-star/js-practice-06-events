# Практична робота 6 (Частина 1): Події - Interactive UI

**Дисципліна:** Основи програмування мовою JavaScript  
**Варіант 1:** Advanced Contact Form

## Опис проєкту
У цьому проєкті реалізовано контактну форму з live-валідацією та прогрес-баром заповнення за допомогою Vanilla JavaScript.

## Реалізований функціонал:
- Live валідація полів (ім'я, email, телефон, тема, повідомлення) в реальному часі.
- Функція **Debounce** для оптимізації перевірки під час вводу.
- Динамічний прогрес-бар, що відображає відсоток правильного заповнення форми.
- Візуальний feedback: зміна кольору рамок, іконки статусу (✓/✗), повідомлення про помилки.
- Лічильник символів для поля "Повідомлення" (до 500 символів).
- Кнопка "Відправити" заблокована до повного та коректного заповнення форми.
- Плавні CSS-анімації та ефект "Floating Labels".
- Імітація асинхронного запиту при відправці (Loading state).

## Технології
- HTML5 (Семантичний)
- CSS3 (Custom Properties, Flexbox, Animations)
- JavaScript (ES6+, DOM Manipulation, Event Delegation)
